# Roadmap

Ideas parked for later, roughly in the order they'd pay off.

## Comic writers

- **Page-turn markers** — LEFT/RIGHT (verso/recto) indicator in each page
  header so reveals can be planned on the turn. Odd = right-hand page. Must
  respect the anchored page numbering (an excerpt starting at PAGE 22 knows
  22 is a left page).
- **House-rule lints** (advisory chips, same style as existing ones):
  em dashes anywhere (hard Sahtu rule), sequence words in panel descriptions
  ("then", "after", "starts to" = one-action-per-panel tell), density
  warnings per page (words > ~120, balloons > 8, panels > 7 without a grid
  note).
- **Character report** — speaking characters with balloon counts and first
  appearance; surfaces name typos (JANIDA vs JANDIA) as duplicate entries.
- **Open-decisions summary** — collect `[bracketed notes]` into a count +
  list so held lines and pending choices don't ship.
- **Balloon-style extensions** — style `(OFF)`, `(WHISPER)`, `(THOUGHT)`,
  `(BURST)` extensions on beat tags (they already parse; render them muted).
- **Art / Letters views** — filtered renders of the same parse:
  descriptions-only for the artist, numbered balloons-only for the letterer.
- **Page map** — thumbnail overview of an issue: panel counts and layout
  badges (splash/grid/spread) per page, for judging rhythm at a glance.

## Screenplay side

- Page numbers on Preview sheets; "N pages" count in the top bar.
- (MORE) / (CONT'D) when a speech breaks across pages.
- Outline sidebar from `#` sections (parser already emits them).
- Character-name autocomplete in the editor.

## App-wide

- Tokened read-only share links (`/s/abc123`) for the webhost move — Preview
  is already the share surface.
- Search across scripts.
- Version history / snapshots per save.
